# Prepositions and register (formality)

## Prepositions
Korean particles do not map one-to-one to English prepositions, so these are
high-error. Learn them as fixed collocations, not by translation.

- "depends to" → "depends **on**"
- "discuss about" → "discuss" (no preposition) or "talk **about**"
- "married with" → "married **to**"
- "good in English" → "good **at** English"
- "arrive to" → "arrive **at/in**"
- "listen music" → "listen **to** music"

## Register (formality)
Match formality to the situation. Casual fillers and slang are fine with friends
but weaken business/academic speech.

Too casual for business:
- "gonna / wanna / kinda / gotta" → "going to / want to / kind of / have to"
- "yeah, nah, yep" → "yes, no"
- "stuff, things" → name the specific thing
- "you know", "like" as fillers → pause instead

Connectors for formal speech: "so" → "therefore / consequently / as a result".

Drill idea: rewrite a casual answer as if speaking to a manager.
